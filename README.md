# 小雅粘粘粘

一个适用于小雅平台的 Tampermonkey / ScriptCat 用户脚本，用于提取任务题目、生成 AI 作答模板、保存作答记录，并辅助查看提交结果。

作者：[Qy](https://qyovo.tech/)

## 功能介绍

- **提取任务题目**：自动读取当前任务中的题干、选项、图片和公式。
- **复制 AI 作答模板**：将题目整理为结构清晰的文本，并自动附带当前课程名，一键复制后即可发送给 AI。
- **保存作答记录**：解析 AI 返回的答案，并按照题目真实 ID 逐题保存。
- **支持多种题型**：支持单选题、多选题、填空题、判断题、简答题、附件题、题组和匹配题。
- **查看提交结果**：任务提交后可查看题目、得分、作答内容、对错状态，以及平台公开的标准答案。
- **题目图片工具**：查看题目图片，并支持复制图片或图片链接。
- **导出图文 PDF**：生成包含题目文字和图片的浏览器打印页面，方便保存为 PDF 后发送给 AI。
- **纯净作业导出**：通过独立抽屉导出不含 AI 提示词的作业 PDF 或 DOC。
- **可选导出正确答案**：可选择在作业导出中加入已确认正确的答案；错误、待批改或无法确认正确的答案不会导出。
- **悬浮操作面板**：提供可拖拽、可折叠的悬浮入口，不影响小雅页面的正常操作。
- **公告与版本提示**：在面板中查看脚本公告和版本更新提示。

| 题型 | 支持情况 |
| --- | --- |
| 单选题 | 支持 |
| 多选题 | 支持 |
| 填空题 | 支持 |
| 判断题 | 支持 |
| 简答题 | 支持多行答案 |
| 附件题 | 识别并跳过保存 |
| 题组 | 支持阅读材料、题干说明与子题展开 |

## 使用方法

1. 安装 Tampermonkey、ScriptCat 或其他兼容的用户脚本管理器。
2. 安装并启用“小雅粘粘粘”脚本。
3. 登录小雅平台并进入需要作答的任务页面。
4. 等待悬浮入口显示题目已同步。
5. 点击“复制题目给 AI”，将内容发送给 AI。
6. 将 AI 返回的答案粘贴到脚本输入框。
7. 点击“保存作答记录”，等待脚本逐题保存完成。

如需导出作业原文，可点击“导出作业”打开独立抽屉，选择 PDF 或 DOC。作业导出不包含 AI 作答提示词。

## AI 返回格式

请让 AI 严格按照以下格式输出答案，不要附加解析或说明：

```text
1 => A
2 => A,C
3 => const | let
4 => 完整的简答题答案
5 => A:a,d | B:b,c
```

- 单选题、判断题：填写一个大写字母。
- 多选题：使用英文逗号分隔多个大写字母。
- 填空题：使用 `|` 分隔多个填空。
- 简答题：题号后的内容会作为完整答案保存，支持多行文本。
- 匹配题：左侧使用大写字母，右侧使用小写字母。

## 图文 PDF

当题目包含较多图片时，可以使用“导出图文 PDF”功能：

1. 点击“导出图文 PDF”。
2. 等待脚本读取并整理题目图片。
3. 在打印窗口中选择浏览器内置的“保存为 PDF / 另存为 PDF”。

使用浏览器内置的 PDF 保存功能，通常可以保留可复制、可搜索的文字内容。

## 纯净作业导出

“导出作业”抽屉支持：

- 选择是否显示课程名和作业名。
- 复制纯作业文本。
- 导出 PDF 或兼容 Word/WPS 的 DOC。
- 可选开启“导出正确答案”。

开启“导出正确答案”后，脚本会优先使用平台公开的标准答案；若平台未公开标准答案，则只导出已经被平台判定正确的用户答案。

## 权限说明

脚本申请以下用户脚本权限：

| 权限 | 用途 |
| --- | --- |
| `GM_setClipboard` | 复制题目文本、图片或图片链接 |
| `GM_xmlhttpRequest` | 在页面请求失败时读取题目图片和公告 |
| `*.ai-augmented.com` | 在小雅平台页面运行并保存作答记录 |
| `*.aliyuncs.com` | 读取题目中使用的图片资源 |

脚本仅在用户主动操作时保存答案，不会自动提交任务。

## 使用提示

- 进入任务页面后，请等待脚本完成题目同步再进行复制或保存。
- 保存答案前建议检查 AI 返回格式和答案内容。
- 附件题不会自动上传附件，也不会提交空答案。
- 平台未公开标准答案时，提交结果面板只展示平台已返回的得分和对错状态。
- “导出正确答案”只导出平台公开或已确认正确的答案，不会猜测未确定答案。
- 如果图片复制失败，脚本会尝试复制图片链接。
